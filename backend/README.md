# Gruber backend

### v.0.1.2

### Backend for Gruber test task


## Installing

`npm install`


## Start backend

For start Gruber backend, you need to define the following variables:

- ClientId

- ClientSecret

- ServerToken

### Example

`ClientId=abcd123 ClientSecret=123abcd ServerToken=dcba321 npm start`
